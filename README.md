# From Indonesia, With Love

A React web application built with Vite that showcases a beautiful "From Indonesia, With Love" welcome experience.

## Project Overview

This is a minimal React + Vite application with a clean, modern interface featuring a gradient background and welcoming message. The app demonstrates best practices for modern web development using React and Vite's fast build tools.

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

```bash
npm install
```

### Development Server

Start the development server:

```bash
npm run dev
```

The application will be available at `http://localhost:5173/`

### Building for Production

Create a production build:

```bash
npm run build
```

The optimized files will be in the `dist/` directory.
